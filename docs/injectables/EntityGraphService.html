Finds the entity network around one or more entities identified by their entity IDs or by the data source codes and record ID's of their constituent records. This attempts to find paths between the specified entities. If no paths exist, then island networks are returned with each island network containing up to a specified number of related entities.

e Array<string | number> No

The multi-valued query parameter where each value is formatted as an EntityIdentifier which may be 64-bit integer entity IDs or JSON-encoded RecordId for the constituent records. The format for all of the provided identifiers must be the same (i.e.: all entity IDs or all data-source / record-id combos).
maxDegrees number Yes

The maximum number of degrees to look for a path between the specified entities.
buildOut number Yes

The maximum number of degrees to build out around each of the specified entities regardless of those entities being on the path between entities.
maxEntities number Yes

The maximum number of entities to return to prevent responses that are too large.

If true, then the 'rawData' field in the response will be a non-null value and contain the additional details.

flag to report request and response progress.
